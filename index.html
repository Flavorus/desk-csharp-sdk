Untyped layer

Connects to the desk.com API. The most flexible approach, but requires a bit of manual work to setup api calls and parse the result.

Strongly typed layer

Connects to the desk.com API, using the untyped layer. Contains mappers and classes that automate all API calls and result parsing.

Tested

All parts of the desk.com C# SDK are tested, to assure that all mappers and parsers work as they should.

Get started

To get started, either
  • grab the source code from GitHub
  • download a certain version from the download page
  • grab the libraries you need directly from within Visual Studio, using NuGet

Once you have the references in place, just follow the instructions in the readme file.

Contribute

Contributions to the SDK are more than welcome. Just send me a pull request, attach the code in an e-mail or whatever. I'd love to have a look at it.

License
